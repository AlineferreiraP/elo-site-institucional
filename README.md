## 🌐 💻 Elo Soluções Digitais

Projeto institucional desenvolvido como desafio prático de Desenvolvimento Web, com aplicação integrada de front-end e back-end.

Toda a estrutura do sistema foi planejada e implementada desde a base, incluindo definição de layout, organização de pastas, lógica de navegação, interatividade e persistência de dados.

#### Tecnologias Utilizadas

HTML5
CSS3
JavaScript
Python
Flask

#### Estrutura do Projeto

O projeto foi organizado de forma simples:

app.py -> arquivo principal da aplicação Flask

contatos.json -> onde os dados do formulário são armazenados

README.md -> documentação

Pasta templates:
Contém todas as páginas HTML do sistema:
index.html, sobre.html, servicos.html e contato.html.

Pasta static; 
Responsável pelos arquivos estáticos do site:

css -> arquivos de estilo

js -> scripts JavaScript

img -> imagens utilizadas no projeto


#### Funcionalidades

Navegação entre páginas utilizando rotas Flask
Layout institucional responsivo
Carrossel de serviços implementado com JavaScript
Destaque automático da página ativa no menu
Formulário de contato com método POST
Armazenamento local dos dados em arquivo JSON

#### ▶️ Como executar o projeto

Abra o terminal na pasta do projeto.
Crie e ative um ambiente virtual.
Instale o Flask.
Execute a aplicação com o comando flask run.
Acesse o endereço informado no terminal (normalmente http://127.0.0.1:5000) no navegador.
